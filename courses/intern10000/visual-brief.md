# 인턴10000 랜딩페이지 시각 디렉션

> Phase 3 입력. `image-prompt-brand` 에이전트가 이 문서를 받아 Midjourney/Sora/Canva 등 도구별 프롬프트를 생성한다. 생성된 이미지는 `public/courses/intern10000/`에 저장.

## 톤 일관성

- 어두운 톤 기조 (다크 차콜 #1e1e1e + 골드 액센트 #d4a64a)
- 자연광 작업 환경, 스튜디오 셋업 ✗
- 진지·직접적, 과장된 미소 ✗
- 인턴의 손·도구·작업 과정 중심 (인물 정면 응시 최소화)
- 도해도/구조 시각화는 핵심 비주얼 요소

## 이미지 목록

### hero.jpg (1920×1080, ★★★ LCP)
- 위치: Hero 섹션 배경 또는 우측 비주얼
- 분위기: 진지·집중·자연광 (어두운 환경에 빛이 떨어지는 작업대)
- 주제: 인턴의 손이 가위와 빗을 들고 마네킹 머리 또는 모델의 두상에 정확한 섹션을 잡는 클로즈업
- 핵심 메시지 연결: "어깨 너머로 배운 기술 vs 구조를 짚는 손"
- 텍스트 오버레이: 없음 (헤드라인은 Next.js에서 처리)

### instructor.jpg (600×800, ★★)
- 위치: Instructor 섹션 (현재 카드 형태 — 추가하려면 카드 상단 또는 좌측)
- 분위기: 진지, 작업 중·도해도 앞에서 설명하는 강사
- 주제: 심일보 강사 — 작업복 또는 어두운 셔츠, 자연광, 측면 또는 3/4 앵글
- 텍스트 오버레이: 없음

### diagram.jpg (1200×800, ★★)
- 위치: Curriculum 섹션 위쪽 또는 Pain Points 결론 callout 옆
- 분위기: 도해도(단면 구조도)와 실제 커트 결과를 나란히 — "구조 이해 → 결과"의 시각화
- 주제: 종이에 그린 단면 구조도(LINE·GRADUATION·LAYER) + 그 옆 실제 마네킹 결과물
- 텍스트 오버레이: 없음 (라벨이 있다면 손글씨 톤)

### workshop-1.jpg (1200×800, ★)
- 위치: Curriculum 또는 Pricing 위쪽
- 분위기: 7명 소수정예 클래스룸 — 학생들이 도해도와 마네킹 사이에서 집중
- 주제: 작업 중인 손·도해도·마네킹·잘려나간 머리카락 — 결과보다 과정
- 텍스트 오버레이: 없음

### workshop-2.jpg (1200×800, ★)
- 위치: Testimonials 섹션 배경 또는 옆
- 분위기: 강사의 1:1 멘토링 장면 — 강사가 학생 옆에서 자세를 교정하는 순간
- 주제: 어깨너머가 아닌 "정면에서 짚어주는" 시각적 대비
- 텍스트 오버레이: 없음

### og-image.jpg (1200×630, ★★★ OG)
- 위치: OpenGraph 미리보기 (페북·카톡 공유)
- 분위기: Hero와 동일한 톤 + 텍스트 오버레이
- 주제: 어두운 배경 + 작업 중인 손 클로즈업 (우측 또는 하단)
- 텍스트 오버레이 (이미지에 포함 OK):
  - 메인: "어깨 너머로 배운 기술은 실력이 되지 않습니다"
  - 서브: "인턴을 위한 만분의 기적 · 14회차 · 정원 7명"
- 모바일에서 가독성 확보 필요 (큰 폰트, 콘트라스트 강)

### favicon / 모바일 홈 아이콘 (선택)
- 정사각 로고 (소옴 브랜드 마크 또는 "인턴10000" 이니셜)
- 어두운 배경 + 골드 액센트


## 검수 체크리스트 (이미지 생성 후)

- [ ] 인물 표정이 과장되지 않은가? (자연스러운 집중·진지함)
- [ ] 톤이 어두운 기조인가? (#1e1e1e 배경과 어울리는가)
- [ ] 손·도구·도해도가 명확히 보이는가?
- [ ] 텍스트 오버레이 비율이 20% 이하인가? (광고용 — OG 제외)
- [ ] OG 이미지: 페북/카톡 미리보기에서 모바일 가독성 확보되는가?
- [ ] 인종·연령 등이 인턴 페르소나(주로 한국, 20~30대 초반)에 맞는가?
