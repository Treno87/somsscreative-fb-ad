## 학습 흐름 요약

```
[기초]    1~2회차  : 이론 + 정확한 자세·섹션·빗질
[LINE]    3~5회차  : Triangle / Square Line + 도해도 + 이론 Level 1·2 테스트
[GRADUATION] 6~8회차 : Triangle / Square / Round Graduation
[LAYER]   9~11회차 : Layer Flat / Convex / Concave
[종합]    12~14회차: Combination + Salon Design (Long·Medium) + 졸업식
```

## 평가 체계

- **이론 테스트 2회**: 4회차(Level 1), 5회차(Level 2)
- **도해도 작성**: 5회차부터 매 회차 (구조 분석 능력 평가)
- **졸업 시험·결과물**: 14회차 Salon Design (Medium)

## 산출물

- 각 회차 도해도 모음 (학생별 14장)
- 졸업 작품 (Salon Design Long / Medium)
- 이론 테스트 점수 (Level 1, Level 2)
