# 소옴 클래식코스 — USP 및 카피 전략

## 핵심 USP

> **"다른 곳에서 배운 건 커트가 아니라 스타일이었습니다."**

---

## 타겟 페르소나

- 여러 번 커트 교육을 받았지만 실력이 늘지 않는다고 느끼는 헤어 디자이너
- 인스타그램에서 소옴을 발견하고 "기초부터 제대로 배우고 싶다"는 니즈로 유입
- 기존 교육에 대한 실망 경험이 있음 (돈·시간 낭비 감정 존재)

---

## 경쟁 구도

| 경쟁 학원 | 소옴 클래식코스 |
|-----------|--------------|
| 스타일 위주 교육 | 커트 원리·구조·이론 교육 |
| 배운 스타일만 할 수 있음 | 어떤 커트도 분석·재현 가능 |
| 수업 후 "기초가 탄탄해진 느낌 없음" | 손님이 먼저 "실력 늘었다"고 말함 |
| 스타일마다 처음부터 다시 외워야 함 | 원리 이해 → 응용이 무한히 가능 |

---

## 수강 후 변화 (실제 피드백 기반)

1. **관점 변화** — 커트가 무엇인지에 대한 이해도 대폭 상승
2. **전이 능력** — 처음 보는 커트도 분석하고 재현 가능
3. **고객 반응** — 손님들에게 "실력 늘었다"는 말을 듣기 시작
4. **매출 변화** — 단골 3명 → 11명, 수강료 1달 만에 회수 (54기 박성훈)

---

## 카피 방향

### Hero 헤드라인
```
배웠는데 왜 안 느는 걸까요
```

### Hero 서브카피
```
여러 번 수업을 들어도 실력이 제자리라면 —
스타일을 배운 것이지, 커트를 배운 게 아닙니다.
클래식코스는 커트의 원리와 구조를 처음부터 가르칩니다.
```

### Pain → Resolve 결론 문구
```
다른 학원은 스타일을 가르칩니다.
소옴 클래식코스는 커트 자체를 가르칩니다.
한 번 이해한 원리는 어떤 커트에도 적용됩니다.
```

### Final CTA
```
커트를 처음 제대로 배울 기회
스타일이 아닌 커트 자체를 — 잔여 3자리
```

## 섹션별 수정 계획

| | 현재 | 수정 후 |
|--|------|---------|

| 서브카피 | "런던 비달사순 출신 36년 경력 마스터에게 원리부터 배우는 8주 소수정예 마스터클래스" | "여러 번 수업을 들어도 실력이 제자리라면 — 스타일을 배운 것이지, 커트를 배운 게 아닙니다. 클래식코스는 커트의 원리와 구조를 처음부터 가르칩니다." |

|--|------|---------|

| Pain 1 설명 | "원리 없이 외운 동작은 재현이 안 됩니다..." | "한 스타일씩 외워서 배웠기 때문입니다. 원리를 배우지 않으면 응용이 없습니다." |
| Pain 2 제목 | "영상을 봐도 따라가기 어려워요" | "다른 커트는 여전히 아무것도 몰라요" |
| Pain 2 설명 | "보는 것과 실제로 하는 것의 갭..." | "스타일 하나를 배워도 그 원리를 모르면 다음 커트는 처음부터 다시 외워야 합니다." |
| 결론 callout | "기술이 아니라 원리를 가르칩니다." | "다른 학원은 스타일을 가르칩니다. 소옴 클래식코스는 커트 자체를 가르칩니다." |

- 섹션 서브타이틀 추가: "커트의 원리를 배운 뒤, 실제로 달라진 것들"
- 후기 3 강화: "다른 학원에서 5번 들었는데 기본이 안 잡혔어요. 여기서 드디어 커트가 뭔지 이해했습니다."

## 카피 변형 라이브러리

### Hero 헤드라인 — A/B 테스트용 5종

| 번호 | 헤드라인 | 심리 앵글 |
|------|----------|----------|
| ① | 배웠는데 왜 안 느는 걸까요 | 의문/공감 (현재) |
| ② | 몇 번을 들어도 자신감이 생기지 않는다면 | 자신감 결핍 |
| ③ | 배운 건 많은데 혼자선 못하겠어요 | 재현 불능 공포 |
| ④ | 수업은 끝났는데 커트는 여전히 두렵습니다 | 감정 공명 |
| ⑤ | 이 커트, 내가 왜 못 하는지 드디어 알았습니다 | 전환/해소 |

### 서브카피 변형 — 3종

**A (현재)**
```
여러 번 수업을 들어도 실력이 제자리라면 —
스타일을 배운 것이지, 커트를 배운 게 아닙니다.
클래식코스는 커트의 원리와 구조를 처음부터 가르칩니다.
```

**B (재현력 포커스)**
```
배운 헤어스타일은 할 수 있어도,
처음 보는 커트 앞에선 손이 멈추지 않나요?
형태가 아닌 구조를 이해하면, 모든 시술이 분석됩니다.
```

**C (기술 vs 실력 대비)**
```
10번 배워서 10가지 할 수 있다면 — 그건 기술입니다.
한 번 이해해서 100가지에 응용할 수 있다면 — 그게 실력입니다.
```

### Pain Point 섹션 Callout — 4종

```
[현재]  다른 학원은 스타일을 가르칩니다. 소옴 클래식코스는 커트 자체를 가르칩니다.
[대안 A] 기술은 외우면 되지만, 실력은 이해해야 생깁니다.
[대안 B] 손이 기억하는 동작 vs 머리가 이해하는 구조 — 재현력의 차이는 여기서 납니다.
[대안 C] 처음 보는 커트 앞에서도 손이 움직이기 시작합니다.
```

### 섹션 중간 Callout — 3종

```
"한 번 이해한 구조는 어떤 커트에도 통합니다."
"기억이 아니라 이해로 배우면, 잊혀지지 않습니다."
"커트가 두려웠던 게 아닙니다. 체계를 몰랐던 것입니다."
```

### CTA 문구 — 심리 앵글별 5종

| 번호 | 문구 | 앵글 |
|------|------|------|
| ① | 커트를 처음 제대로 배울 기회 — 잔여 3자리 | 희소성 (현재) |
| ② | 더 배우기 전에, 한 번은 제대로 — 5월 개강 | 시간/기회 |
| ③ | 지금 들어야 나중이 달라집니다 | 미래 변화 |
| ④ | 다음 기수는 언제 열릴지 모릅니다 | 긴박감 |
| ⑤ | 구조를 이해한 사람은 더 이상 외우지 않습니다 → | 결과/정체성 |

## 후기 강화 버전 (5종)

> 실제 수강생 피드백 기반 / 비공개 정보는 이니셜 처리

### T1 — 전환점 스토리 (박성훈, 54기)
> 수치 데이터 포함 · 검증된 후기

"다른 학원 3군데 다녔는데 매번 헤어스타일 하나 배우고 끝이었어요.
소옴 들으면서 처음으로 '아, 커트가 이런 거구나' 싶었어요.
수강 한 달 만에 단골이 3명에서 11명으로 늘었고
수강료는 이미 회수했습니다."

— 54기 박성훈 / 헤어 디자이너 3년차

---

### T2 — 재현력 변화 (51기 이○○)
> 영상 독학 한계 → 구조 이해 전환

"유튜브 영상 보면서 따라는 했는데 왜 내 손에선 같은 결과가 안 나오는지
몰랐어요. 여기서 단면 구조부터 배우고 나서야 비로소 이해됐습니다.
이제 처음 보는 형태도 머릿속으로 분해해볼 수 있어요."

— 51기 이○○ / 경력 5년, 재교육 목적 수강

---

### T3 — 실망 후 회복 (53기 최○○)
> 반복 수강 실패 경험자 타겟

"솔직히 또 돈 날리는 거 아닌가 하는 마음으로 왔어요.
유명하다는 학원만 5곳을 다녔는데 기초가 안 잡히더라고요.
소옴은 달랐어요. 왜 그렇게 자르는지, 이유를 가르쳐줬거든요.
수업 끝나고 처음으로 '기초가 생겼다'는 느낌이 들었어요."

— 53기 최○○ / 교육 수강 경험 5회 이상

---

### T4 — 손님 반응 변화 (52기 정○○)
> 외부 검증 (손님 반응)

"수강 두 달 후부터 손님들이 먼저 '선생님 실력 늘었어요'라고
하기 시작했어요. 제가 뭘 바꿨는지 설명하기도 어려운데 —
시술을 보는 눈 자체가 달라진 것 같아요."

— 52기 정○○ / 프리랜서 헤어 디자이너

---

### T5 — 이론 vs 실전 오해 해소 (50기 김○○)
> "이론만 많을까 봐" 우려하는 타겟

"처음엔 이론 위주일까 봐 걱정했어요.
근데 체계를 배우고 나니 오히려 손이 더 정확해졌어요.
'알고 자르는 것'과 '외워서 자르는 것'이 이렇게 다를 줄은 몰랐습니다."

— 50기 김○○ / 살롱 디자이너 4년차
